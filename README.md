# 工具类

常用工具类整合，添加commons-lang3、guava、Jackson等常用工具。

pom文件中添加依赖。

	<dependency>
        <groupid>net.hzbox.vj</groupid>
        <artifactid>vj-utils</artifactid>
        <version>1.1.1</version>
    </dependency>
